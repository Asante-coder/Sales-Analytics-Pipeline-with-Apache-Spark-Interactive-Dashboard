**# Project Title**

Sales Analytics Pipeline with Apache Spark & Interactive Dashboard

🎯 **## Project Goal**

Build a complete end-to-end data pipeline that:

* Ingests raw sales data
* Cleans and transforms it
* Performs business analysis
* Stores optimized outputs
* Powers an interactive dashboard

🧰 ## **Tech Stack**

* Apache Spark
* PySpark
* Pandas
* Streamlit
  
🪜 Step-by-Step Build Guide

🔹 STEP 1: Project Setup
📁 Create folder structure
sales-analytics-spark/

Inside:

* data/raw/
* data/processed/
* src/
* notebooks/
* dashboard/
* outputs/
